# Code-Debugger

AI Code Debugger lets you upload a screenshot of any code error and instantly get AI-generated insights — either guided hints to help you learn, or a complete fix with explanation. Powered by Google Gemini and built with Streamlit, it's designed to be fast, intuitive, and educational.

✨ Features
📸 Screenshot Upload — Supports PNG, JPG, and JPEG formats
💡 Hints Mode — Get guided clues to find the bug yourself (great for learning)
⚡ Full Solution Mode — Get the complete corrected code with step-by-step explanation
📋 Detailed Debug Report — Covers error type, root cause, fix, and best practices
📥 Download Report — Save the AI analysis as a Markdown file
🎨 Clean UI — Minimal, user-friendly interface built with Streamlit

Usage

Upload a screenshot of your code error using the file uploader.
Choose a debug mode:
💡 Hints Only — Receive guided clues without spoiling the answer.
⚡ Full Solution + Code — Get the complete fix and explanation.
Click 🚀 Debug My Code to trigger the AI analysis.
Review the AI Debug Report rendered in the app.
Optionally download the report as a .md file.

Tech Stack
Web UI framework - Streamlit
AI model - GEMINI FLASH 3.0 
API - GEMINI API
Image processing - Pillow
Environment variable management - Python dotenv
Language - Python
